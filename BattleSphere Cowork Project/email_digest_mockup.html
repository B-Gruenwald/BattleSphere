Design notes (not part of email)

  • Platform news block — only shown when you have queued announcements. Can be removed entirely if none pending.
  • Commander's Orders — only shown when an organiser has queued a message. Subtle gold border distinguishes it from automated content.
  • Bulletin — shows dispatch number, title, first ~40 words, link to full dispatch on site.
  • Current Events — date blocks use dark BattleSphere header colour to echo the site. Up to 3 events shown; if more, a "View all events →" link.
  • Second campaign — if user is a member of multiple campaigns, each gets its own section with the same structure, separated by a thin rule.
  • Footer — "Manage preferences" links to /profile/notifications (page we'll build). Unsubscribe uses a one-click token link.
